The Best Windows Software

Created: 2020-01-29 | Modified: 2020-01-29

This page lists the best software for Windows, more specifically Windows 10.

Image Viewer

  • Windows Image Viewer
    • Enabled in Windows 10 through a registry key
    • It displays images
    • You need nothing else

Screenshot

  • Snipping Tool
    • It just works, and it ships by default in Windows

Image Editor

  • Gimp
    • It also just works

Media Player

  • mpv
    • Just a media player. No excess.
  • VLC
    • Super fully featured
    • Somewhat "Bloated"
    • Can convert between audio and video formats, watch streams, and more.

Music Player

  • Foobar2000
    • Super customizable
    • Plays any audio type
    • Extensible

eBook Reader

  • Calibre
    • A library from your books in your computer

File Sharing

  • qBittorrent: This will fill your torrenting needs.
  • Soulseek: No comment, but it's the best ;).

Text Editor

  • gVim
    • It would be console vim, but ALL Windows terminals are laggy garbage, so gVim it is
    • You don't need anything else for text editing/note taking

Keybinding System

  • AutoHotkey
    • It's a feature full scripting language for binding keys to actions
    • Good for automation

Unix System

    WSL
    • There's no point in using Cygwin because of WSL
    • The only issue with WSL is that all terminals for Windows (PowerShell, cmd, Windows Terminal, Alacritty, etc.) are laggy trash.
    • You can't do any text editing or anything that requires a semblance of responsivity

Browser

  • Firefox
    • The best

Mathematics

  • Python3
    • (This is personal preference)
    • Allows me to write quick functions
  • Mathematica
    • This is good too
    • I don't do in-depth Mathematics on the daily, therefore, I don't use it
